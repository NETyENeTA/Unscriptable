# Проект Unscriptable

Unscriptable — это образовательная платформа на .NET 8, ориентированная на аналитику успеваемости.

ТЕХНОЛОГИИ
- Backend: ASP.NET Core 8.0
- БД: PostgreSQL
- Инструменты данных: EF Core + Dapper
- Безопасность: JWT & Cookies
- Тесты: xUnit + Moq
- CI: GitHub Actions

АРХИТЕКТУРА
Проект следует принципам Чистой Архитектуры:
- Domain: Сущности и логика.
- Application: Интерфейсы и DTO.
- Infrastructure: БД, миграции, SQL-отчеты.
- API: Эндпоинты и безопасность.

ОТЧЕТЫ
Реализовано 8 модулей отчетности, включая рейтинги студентов, тренды групп,
личный прогресс и статистику активности по дням недели.

БЕЗОПАСНОСТЬ
- Авторизация на основе ролей.
- Студенты имеют доступ только к своим данным.
- Менеджеры имеют полный доступ к аналитике.

РАЗВЕРТЫВАНИЕ
1. Настройте PostgresConnection в appsettings.json.
2. Выполните: dotnet ef database update (для обновлении бд и создания её)
3. Запуск: dotnet run
